# Focus Flow
Focus Flow is a simple task manager designed to help users overcome procrastination, stay organized, and track their tasks easily.
## Features
- You can add tasks and sort them with name, priority and due date
- View your saved tasks in the progress page
- Delete your completed tasks easily
- Tasks are are saved so you can go back to them
## Pages
1. Home
   - Overview of procrastination and productivity tips
   - YouTube video for help
2. Task manager
   - Add your tasks
3. Progress
   - View your saved tasks
   - Delete tasks
##Languages
- HTML
- CSS
- JavaScript
## Steps
1. Open the page
2. Go to add task to create tasks
3. View your tasks in progress
4. Tasks will stay in your browser until they're deleted
